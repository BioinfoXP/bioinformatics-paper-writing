# Writing Flow Guide

## Goal

Improve paragraph-level clarity, sentence-to-sentence coherence, and section logic without diluting scientific precision.

## When to Use

Use this file when the user asks whether writing flows, makes sense, feels disconnected, or needs better transitions.

## Fast Flow Check

1. Read as an external reader:
   - Is there one visible message in the paragraph?
   - Does the first sentence state that message?
   - Are key nouns understandable without hidden context?
   - Does each sentence connect by cause, contrast, consequence, refinement, or validation?
2. Reverse-outline the section:
   - write the local thesis,
   - write each paragraph topic sentence,
   - write the evidence used in each paragraph,
   - check whether evidence supports the topic sentence and whether the topic sentence supports the section claim.

## Revision Moves

1. Move the paragraph claim into the first sentence.
2. Define cohorts, signatures, cell states, and endpoints before reusing them.
3. Split mixed-purpose paragraphs into discovery, validation, or interpretation.
4. Add light transition language only where the logical relation is otherwise unclear.
5. Delete sentences that repeat method detail without advancing the claim.

## Useful Transition Types

1. Cause or consequence:
   - `therefore`
   - `accordingly`
   - `as a result`
2. Contrast:
   - `however`
   - `by contrast`
   - `in contrast`
3. Refinement:
   - `more specifically`
   - `notably`
   - `consistent with this`
4. Validation:
   - `this pattern was further supported by`
   - `we next validated this finding in`

## Weak Patterns to Fix

1. A paragraph starts with method detail instead of the finding.
2. One paragraph mixes observation, mechanism, and clinical implication.
3. Sentences introduce unexplained abbreviations or group labels.
4. The final sentence does not bridge to the next question.

## Checklist

1. One paragraph, one message.
2. First sentence states the message.
3. Evidence follows in a logical order.
4. Interpretation does not outrun the evidence.
5. The final sentence either closes the point or opens the next step.
