# Validation and Supplementary Analyses Guide

## Goal

Strengthen the paper beyond a first discovery by proving robustness, reproducibility, and translational discipline.

## Validation Layers

Prefer several of these layers when possible:

1. Internal validation within the study design.
2. Independent external cohort validation.
3. Orthogonal assay validation.
4. Functional or perturbation support.
5. Sensitivity analysis for thresholds, covariates, or feature choices.

## Prognostic or Predictive Model Checklist

1. Separate discovery from validation cohorts.
2. State endpoint definitions clearly.
3. Report discrimination and, when relevant, calibration.
4. Compare against standard clinicopathologic variables or existing signatures.
5. Avoid calling a model clinically actionable without strong external support.

## Robustness Analyses

Useful analyses include:

1. multivariable adjustment,
2. subgroup consistency,
3. alternative cutoff strategies,
4. batch or cohort sensitivity,
5. ablation of feature families,
6. model simplification tests.

## Interpretation Rule

Validation should strengthen a claim, not inflate it.
If validation is partial, say so explicitly.

## Checklist

1. What is discovery only?
2. What is validated independently?
3. What remains correlative?
4. What remains hypothesis-generating?
